# Full Stack Date/Time & Name App

A simple full-stack application built with **React + Vite** (frontend) and **Node.js + Express** (backend).

## Features
- Displays current **date** and **time** (live updates every second)
- Asks **"What is your name?"**
- On submit, displays **"Your name is [name]"**

## How to Run

### 1. Install backend dependencies
```bash
cd backend
npm install
```

### 2. Start the backend server
```bash
cd backend
npm start
```
Backend runs on `http://localhost:5000`

### 3. Install frontend dependencies (in a new terminal)
```bash
cd frontend
npm install
```

### 4. Start the frontend dev server
```bash
cd frontend
npm run dev
```
Frontend runs on `http://localhost:3000`

Open **http://localhost:3000** in your browser.
